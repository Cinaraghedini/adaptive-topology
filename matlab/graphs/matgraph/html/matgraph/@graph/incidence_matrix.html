incidence_matrix

PURPOSE

incidence_matrix(g) --- return the vertex/edge incidence matrix.

SYNOPSIS

function M = incidence_matrix(g)

DESCRIPTION 

 incidence_matrix(g) --- return the vertex/edge incidence matrix.
 We return M an nv-by-ne matrix whose ij entry is 1 if vertex i is an
 end point of edge j.
