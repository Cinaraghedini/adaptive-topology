SYNOPSIS

function circulant(g,n,k)

DESCRIPTION 

 circulant(g,n,k) --- overwrite g with an n,k circulant graph
 g has n vertices {1,2,...,n}. Each vertex has an edge to the next k in
 line. So this is a 2k regular graph.
