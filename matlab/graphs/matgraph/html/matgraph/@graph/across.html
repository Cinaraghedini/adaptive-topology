across

PURPOSE

across(g,X,Y) --- find all edges with one end in X and other end in Y.

SYNOPSIS

function elist = across(g,X,Y)

DESCRIPTION 

 across(g,X,Y) --- find all edges with one end in X and other end in Y.
 We require X and Y to be disjoint. If Y is omitted, then Y = V(g)-X.
 We do not require X U Y = V(g).
